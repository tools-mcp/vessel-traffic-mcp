# Marketing Assets

Store launch screenshots and short clips here. Do not store API keys, private
captures, cookies, HAR files, logged-in dashboards, or provider account pages.

Recommended files:

- `map-ui-ever-given.png` - desktop screenshot of the local map UI after
  searching `EVER GIVEN`.
- `map-ui-ever-given-mobile.png` - optional narrow viewport screenshot.
- `mcp-client-source-attribution.png` - MCP client response showing
  `source.provider` and `source.landingUrl`, with no secrets.

Before publishing assets, verify that:

- Source provider and URL are visible.
- No browser cookies, account names, API keys, or tokens are visible.
- Coordinates, timestamps, and caveats are readable.
- The screenshot reflects public or fixture-backed data only.
